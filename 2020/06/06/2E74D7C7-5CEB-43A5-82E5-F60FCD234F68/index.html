命令行启动安卓模拟器

Posted on 2020-06-06 | Edited on 2020-06-11

要使用到SDK里面的emulator文件夹下的emulator命令文件

${android-sdk-macosx sdk目录path}/emulator

罗列所有可用的模拟器

./emulator -list-avd

启动对应的模拟器

./emulator -avd 模拟器名称

PANIC: Missing emulator engine program for ‘x86’ CPU. 是因为emulator使用不对，需要是emulator文件夹下的emulator才对

将emulator配置为环境变量

  1. 编辑~/.zshrc
  2. export PATH
  3. source ~/.zshrc
